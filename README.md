# PersonaMail AI

PersonaMail AI is an AI-powered personalized email generation platform built using:

- FastAPI
- Streamlit
- Ollama
- Llama3

The application generates contextual, professional, and human-like emails dynamically based on user demographics and email requirements.

---

# Features

- AI-powered email generation
- Dynamic tone adaptation
- Personalized contextual emails
- Subject line generation
- Multiple email use cases
- FastAPI backend
- Streamlit frontend
- Ollama local LLM integration

---

# Tech Stack

Frontend:
- Streamlit

Backend:
- FastAPI

AI:
- Ollama
- Llama3
